All In One VAT Reports

This Module is developed to enhance the functionality of VAT Reporting in Odoo Community Version. Reconciliation of VAT Reporting is made Easy for Accountants. It is a general Purpose Application and can be used in any Country for reporting.In this Application we have Added features of:

  1. Summarized VAT Reporting for Each Types of Taxes. We have automated the Tax Refundable and Payable Calculation for each VAT Period , with advance filters of Adjustments
  2. VAT Statement That show all customer related and Invoicing Related information
  3. Individual Taxes and their base Amount for Filling returns
  4. With advance filtration option by Tax Group
  5. Tax Label and Tax Scope
  6. VAT Payment Functionality
  7. VAT Refunds Functionality
  8. VAT Other Misc. Adjustments
